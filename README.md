# Simaptics

**Simple Map Tactics**

Simaptics is a personal project where I mix strategy, maps, tooling, and infrastructure into one evolving platform.  
It’s a playground for ideas that sits somewhere between game development and real-world systems engineering.

This project combines a bunch of things I enjoy:

- 🎮 Video games  
- 💻 Coding  
- ☁️ DevOps  
- 🛠️ Creating new things  
- 🎲 TTRPGs  

---

## 🚧 Work in Progress

This is an active, evolving project:

- Some parts are rough
- More features and services are coming
- Optimization is ongoing
- Expect occasional downtime while things change
